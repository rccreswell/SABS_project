# SABS_project



To test the web app, use the following instructions.

First set up flask

Linux:

```
$ export FLASK_APP=sabs_pkpd
```

Windows:

```
> set FLASK_APP=sabs_pkpd
```

Then install the app


```
$ pip install -e .

$ flask run
```
